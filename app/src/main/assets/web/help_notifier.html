Notifier

A JavaScript interface through which you can send push notifications. It is recommended to use only in the event of OnReceive()

Example:
if(value>100)Notifier.push("Alarm!!!"); if(value<90)Notifier.stop();

Please note, you must call the stop() method in the same script of the widget, otherwise the message will come every time when you start the application. Also make sure that before removing the widget from the dashboard, last was caused by Notifier.stop()

See also: OnReceive
